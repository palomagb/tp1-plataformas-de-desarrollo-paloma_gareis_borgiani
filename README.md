# Pet Care Tracker

Aplicación web interactiva para gestionar y organizar los cuidados diarios de tus mascotas. 
Este proyecto forma parte del **Trabajo Práctico 1** de la materia **Plataformas de Desarrollo** - Escuela Da Vinci.

## Funcionalidades
- **Gestión de cuidados:** Permite añadir y eliminar tareas de cuidado.
- **Estado de tarea:** Posibilidad de marcar tareas como completadas.
- **Buscador:** Filtrado en tiempo real para encontrar cuidados específicos rápidamente.
- **Persistencia:** Los datos se guardan en el navegador mediante `localStorage`, por lo que no se pierden al recargar.
- **Validación:** Sistema de notificaciones (Toast) para evitar tareas duplicadas.

## 🛠️ Tecnologías utilizadas
- **HTML5** (Estructura semántica).
- **JavaScript** (Lógica, DOM y Eventos).
- **Tailwind CSS** (Diseño responsivo y moderno).
